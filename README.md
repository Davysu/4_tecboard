# 4_tecboard

Projeto de exemplo (HTML + CSS) — pasta `4_tecboard`.

Conteúdo:
- `index.html`
- `css/style.css`
- `img/` (imagens)

Como publicar:
1. Inicialize um repositório Git nesta pasta: `git init`
2. Comite os arquivos: `git add . && git commit -m "Initial commit"`
3. Crie o repositório remoto no GitHub e dê push. Exemplo com GitHub CLI:
   `gh repo create 4_tecboard --public --source=. --remote=origin --push --confirm`

Se preferir, crie o repositório manualmente no GitHub e execute:
```
git remote add origin https://github.com/SEU_USUARIO/REPO.git
git branch -M main
git push -u origin main
```
